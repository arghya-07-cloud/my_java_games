# Snake Game (Java)

A simple classic Snake Game built in Java using the Swing library. The snake grows each time it eats food, and the game ends when the snake hits a wall or its own body.

## How to Play
- Use Arrow Keys to move the snake:
  - Up, Down, Left, Right
- Eat the red food to increase your length.
- Avoid hitting the walls or your own tail.
- When you collide, the game shows “Game Over!”.

## Requirements
- Java JDK 8 or higher
- Works on Windows, Linux, and Mac

## How to Run
1. Download the file: SnakeGame.java  
2. Open a terminal/command prompt in the file folder  
3. Compile:  
   javac SnakeGame.java  
4. Run:  
   java SnakeGame
